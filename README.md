# Laboratório 1 - Atividade - Dispositivos Móveis

Aplicativo Android desenvolvido para a disciplina de Dispositivos Móveis que permite ao usuário inserir nome e idade em uma primeira tela e enviar essas informações para uma segunda tela, onde os dados são exibidos. O aplicativo também realiza testes do ciclo de vida das Activities.
